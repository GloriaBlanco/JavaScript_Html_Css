DISPLAYS

o DISPLAYé um apropriedade CSS qye permite controlar melhor a visualização de um elemeto dentro da página, neste primeiro momento iremos trabalhar com as 3 variaveis iniciais desta propriedade:

  • Block: vertical, elemento em bloco, este ocupa 100% da largura disponível e sempre está iniciado em uma nova linha. Ex: divs, titulos do h1 até h6, p,
  • InLine: horizontal, elemento em linha, no qual ocupa apenas o espaço necessário para exibir o seu conteúdo. Ex: links
  • InLine-Block: elemento de blocos em linha, onde ira permitir que seja alterado tanto largura quano altura dos seus elementos. Ex. menu
